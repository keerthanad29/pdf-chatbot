# PDF Q&A Chatbot

A chatbot that answers questions from uploaded PDF documents.

## Features
- Upload any PDF file
- Ask questions in natural language
- Get accurate answers from the document

## Tech Stack
- Python
- Streamlit
- LangChain
- Groq (LLaMA 3.1)
- TF-IDF + Cosine Similarity

## How to Run
1. Clone the repo
2. Install dependencies: `pip install -r requirements.txt`
3. Add your Groq API key in `.env` file
4. Run: `streamlit run app.py`
